iOS 网络日志拦截工具 - 文件清单
================================================================================

📦 核心功能文件（需要添加到 Xcode 项目）
================================================================================

1. NetworkInterceptor.swift
   - 网络拦截核心类
   - 基于 URLProtocol 实现
   - 拦截所有 URLSession 请求

2. InterceptedRequest.swift
   - 拦截请求数据模型
   - 存储请求和响应信息
   - 提供数据格式化方法

3. NetworkInterceptorManager.swift
   - 拦截管理器（单例）
   - 启动/停止拦截
   - 管理和过滤记录

4. FloatingButton.swift
   - 可拖拽的悬浮按钮
   - 支持点击隐藏、长按显示日志
   - 自动吸附到屏幕边缘

5. NetworkLogViewController.swift
   - 网络日志列表页面
   - 搜索和过滤功能
   - 清空和导出功能

6. NetworkLogCell.swift
   - 日志列表单元格
   - 显示请求方法、状态码、URL、耗时

7. NetworkLogDetailViewController.swift
   - 日志详情页面
   - 7 个标签页展示完整信息
   - JSON 自动格式化

步骤 1: 添加核心文件到 Xcode 项目

================================================================================
🎯 使用方法
================================================================================

1. 点击悬浮按钮 → 隐藏按钮
2. 长按悬浮按钮 → 显示网络日志列表
3. 拖拽悬浮按钮 → 移动到任意位置（自动吸附边缘）
4. 点击日志列表项 → 查看详细信息
5. 使用搜索框 → 搜索 URL
6. 使用过滤器 → 按方法或状态过滤
7. 点击清空按钮 → 清空所有日志
8. 点击导出按钮 → 导出为 JSON

================================================================================
⚠️ 注意事项
================================================================================

1. 建议仅在开发/测试环境使用
2. 大量请求会占用内存，定期清空日志
3. 日志可能包含敏感信息，注意隐私安全
4. 如需上架 App Store，使用编译条件控制：
   #if DEBUG
   NetworkInterceptorManager.shared.startIntercepting()
   #endif
