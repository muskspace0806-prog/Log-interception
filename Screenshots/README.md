# ZWB_LogTap 截图

## 如何添加截图

请将以下截图保存到此目录：

1. **main_screen.png** - 主界面（显示两个测试按钮和悬浮按钮）
2. **http_list.png** - HTTP 请求列表（显示成功和失败的请求）
3. **http_detail.png** - HTTP 请求详情（显示完整的请求和响应信息）
4. **im_list.png** - WebSocket 消息列表（显示各种类型的消息）
5. **im_detail.png** - WebSocket 消息详情（显示消息内容）

## 截图要求

- 格式：PNG
- 设备：iPhone（推荐 iPhone 14 Pro 或更新）
- 方向：竖屏
- 内容：确保包含成功和失败的请求示例

## 如何截图

### 模拟器
1. 运行应用
2. 按 `Cmd + S` 保存截图
3. 截图会保存到桌面

### 真机
1. 运行应用
2. 同时按 `电源键 + 音量上键`
3. 从相册导出截图

## 注意事项

- 确保截图中不包含敏感信息
- 使用真实但安全的测试数据
- 截图应清晰展示功能特性
