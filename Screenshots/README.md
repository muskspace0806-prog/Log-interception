# Screenshots

请在这个目录下添加以下截图：

## 需要的截图

1. **floating_button.png** - 悬浮按钮截图
   - 显示应用主界面和右下角的悬浮按钮
   - 建议尺寸：750x1334 或 1125x2436

2. **http_list.png** - HTTP 日志列表截图
   - 显示 HTTP 请求列表
   - 包含搜索框和过滤器
   - 建议尺寸：750x1334 或 1125x2436

3. **http_detail.png** - HTTP 详情页面截图
   - 显示请求详情的某个标签页
   - 展示 JSON 格式化效果
   - 建议尺寸：750x1334 或 1125x2436

4. **websocket.png** - WebSocket 消息截图
   - 显示 IM 消息列表或详情
   - 展示消息内容
   - 建议尺寸：750x1334 或 1125x2436

## 如何截图

1. 在模拟器或真机上运行应用
2. 使用 Cmd+S (模拟器) 或 电源键+音量上键 (真机) 截图
3. 将截图重命名并放到这个目录

## 注意事项

- 使用高清截图
- 确保截图内容清晰可读
- 避免包含敏感信息
- 可以使用工具添加设备边框（如 [Screenshot Framer](https://www.screenshotframer.com/)）
